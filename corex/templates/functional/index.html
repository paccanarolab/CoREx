Human functional modules for drug repurposing against SARS-CoV-2

Mateo Torres, Suzana Santos, Diego Galeano, María del Mar Sánchez, Alberto Paccanaro

We present FDA approved drugs and their targets in the context of human functional modules containing SARS-CoV-2 host proteins [1] – these are 332 human proteins that have been experimentally identified to bind to viral proteins during infection.

What we did:

  1. We predicted functional similarities between proteins by combining five types of experimental interactions (neighborhood, coexpression, experiments, database, textmining) from the STRING database [2] together with information from InterPro [3].
  2. We then created a human functional interactome, where each node is a protein and two proteins are connected if they are predicted to be functionally related.
  3. We identified functionally coherent modules by clustering this network using ClusterONE [4].
  4. Finally, we selected functional modules that include at least one SARS-CoV-2 host protein and one drug target retrieved from DrugBank [5].

Selecting a drug from the list below will take you to a page containing the functional modules that are hit by that drug and contain at least one SARS-CoV-2 host proteins.

For each functional module we include:

  1. A list of drugs that target proteins in the same module with their respective ATC categories.
  2. Connectivity Map scores for each drug from the Broad Institute [6] when available.
  3. For each drug, information on whether it is currently on clinical trials from ClinicalTrials.gov when this is available.
  4. GO functional enrichment for every predicted module, calculated using Enrichr [7].
